## TASK 2.1.7 [задание на stepik](https://stepik.org/lesson/308885/step/7?unit=291011)
Заполнить таблицу **author**. В не` включить следующих авторов:

- Булгаков М.А.
- Достоевский Ф.М.
- Есенин С.А.
- Пастернак Б.Л.

**Решение**:

```SQL
INSERT INTO author (name_author)
VALUES
   ('Булгаков М.А.'),
   ('Достоевский Ф.М.'),
   ('Есенин С.А.'),
   ('Пастернак Б.Л.');

SELECT * FROM author;
```

**Результат**:

```SQL
+-----------+------------------+
| author_id | name_author      |
+-----------+------------------+
| 1         | Булгаков М.А.    |
| 2         | Достоевский Ф.М. |
| 3         | Есенин С.А.      |
| 4         | Пастернак Б.Л.   |
+-----------+------------------+
Affected rows: 4
```
